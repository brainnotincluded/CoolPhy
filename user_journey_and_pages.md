# User Journey & All Platform Pages (English)

## Main User Flow
1. Home — Entry point, intro, latest news, quick links to subject areas, and login/registration options.
2. Login — User login via email/password or OAuth (Google, VK, etc), access to personal dashboard after login.
3. Registration — New user registration, email confirmation, selection of subjects.
4. Password Recovery — Forgot password process, reset via email link.
5. Lecture Catalog — List of all lectures; subject, topic & tag filters; quick search.
6. Lecture Page — View single lecture, LaTeX-format content, mark as studied, add notes, see related problems.
7. Problem Catalog — Browse/search problems, filter by subject, topic, difficulty, tags, and solved/unsolved status.
8. Problem Page — Problem statement (LaTeX), answer submission, solution, check status and receive AI feedback.
9. Topics Tree — Hierarchical view of all topics, fast navigation to problems and lectures by subject (math/physics/informatics).
10. Topic Page — Overview of topic, list of related problems and lectures, personal statistics and recommendations.
11. Professor Chat (LLM) — AI chat assistant for discussing lectures/problems, guidance and answer checking.
12. Profile/Dashboard — User stats, progress, total points, badges, chart visualizations; personal performance overview.
13. Profile Settings — Edit profile, manage privacy, change password, update subject interests and integrations.
14. Activity History — Timeline of solved problems, read lectures, interactions with AI professor.
15. Notifications — Site alerts: new tasks, messages, content updates, achievements.
16. Global Search — Search among lectures, problems and topics with autofiltering.
17. Help/FAQ — Platform help, instructions for new users, frequently asked questions.
18. Leaderboard — Top users ranking by points and achievements; detailed stats.
19. Admin Panel (admins only) — Manage users, lectures, problems, topics; platform moderation & analytics.
20. Lecture/Content Editor (admin) — Author/edit lectures, content & tags, LaTeX/Markdown support.
21. Problem Editor (admin) — Create/edit problems, solutions, hints, scores, topic linking.

---

## Each Page Description:

- **Home:** The main entry page with general platform info, news, and navigation shortcuts
- **Login:** Authentication interface for registered users
- **Registration:** User signup form; choose subjects and set up profile
- **Password Recovery:** Forgot password flow for account access recovery
- **Lecture Catalog:** Directory of all platform lectures for study and navigation
- **Lecture Page:** Detailed view of a lecture; LaTeX format, notes, and status actions
- **Problem Catalog:** Overview and filterable list of all problems/tasks
- **Problem Page:** Full details of a problem, interactive solving and answer check
- **Topics Tree:** Visual navigation for all subject/curriculum topics
- **Topic Page:** Topic summary, linked educational content, personal progress stats
- **Professor Chat:** AI assistant for learning support and Q&A
- **Profile/Dashboard:** User’s learning stats, achievements, cumulative progress
- **Profile Settings:** Change account, privacy, integrations, subject interests
- **Activity History:** Chronological log of user events within the platform
- **Notifications:** Alerts for new content, solutions, achievements, messages
- **Global Search:** Search for any lectures/problems/topics with auto-suggestions
- **Help/FAQ:** Guide to platform features, common troubleshooting, and rules
- **Leaderboard:** Ranking of top users according to points/achievements
- **Admin Panel:** Platform management for content, users, moderation and stats
- **Lecture/Content Editor:** Tool for adding/editing lectures, tags and connections
- **Problem Editor:** Tool for authoring/editing problems, setting scores, solutions

---

## Notes:
- Most pages are available to any user, except admin panel and editors (restricted).
- Layout and navigation are optimized for fast switching: dashboard links, search, notifications.
- Designed for extensibility if new subject areas or features are needed in future.
