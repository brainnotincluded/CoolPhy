TikZ Diagram Layout Control

You can control the width, alignment, and height of TikZ diagrams directly in your LaTeX code using meta comments.

Basic Syntax

Place a comment on the line immediately before your \begin{tikzpicture}:

% tikz-preview: width=50%, align=center

\begin{tikzpicture}
  \draw (0,0) -- (2,0);
  \draw (2,0) circle (0.3cm);
\end{tikzpicture}

Parameters

Parameter Values Description
width 20‚Äì100 (percent) Width of the diagram container relative to page width
align inline, left, center, right Horizontal alignment of the diagram
height Number (pixels) Fixed height for the diagram (optional, defaults to auto)

Examples

Full-width centered diagram
% tikz-preview: width=100%, align=center

\begin{tikzpicture}
  \draw (0,0) rectangle (4,3);
\end{tikzpicture}
Side-by-side diagrams
% tikz-preview: width=48%, align=inline

\begin{tikzpicture}
  \draw (0,0) circle (1cm);
  \node at (0,0) {Diagram A};
\end{tikzpicture}

% tikz-preview: width=48%, align=inline

\begin{tikzpicture}
  \draw (0,0) circle (1cm);
  \node at (0,0) {Diagram B};
\end{tikzpicture}

Use align=inline to place diagrams on the same row

Fixed height diagram
% tikz-preview: width=60%, align=center, height=200

\begin{tikzpicture}
  \draw (0,0) -- (5,0) -- (5,3) -- (0,3) -- cycle;
\end{tikzpicture}

Useful for maintaining consistent spacing

Note: Meta comments are automatically hidden in the preview and PDF export. They only affect layout and do not appear in the rendered output.

Features

Hover to Highlight

Hover over any formula, section, or TikZ diagram in the preview to highlight the corresponding LaTeX code in the editor.

iPad & Apple Pencil Support

In the TikZ Visual editor, touch input is disabled by default (palm rejection). Toggle Touch: Off/On in the toolbar to enable finger drawing.

Tips & Tricks

Multiple diagrams in a row

Set width to less than 50% and use align=inline for each diagram:

% tikz-preview: width=30%, align=inline
\begin{tikzpicture}...\end{tikzpicture}

% tikz-preview: width=30%, align=inline
\begin{tikzpicture}...\end{tikzpicture}

% tikz-preview: width=30%, align=inline
\begin{tikzpicture}...\end{tikzpicture}

Controlling spacing

Use the height parameter to reserve vertical space and prevent layout shifts as TikZ compiles.

Default behavior

If no meta comment is provided, diagrams default to:

  • width: 100%
  • align: inline
  • height: auto
